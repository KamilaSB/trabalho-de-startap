# trabalho-de-startap

Sobre o ProjetoO projeto consiste na idealização de uma plataforma digital voltada para a comunidade de jogadores de RPG e jogos de tabuleiro. O objetivo central é solucionar a fragmentação de ferramentas, facilitando a formação de grupos e a organização de partidas, sejam elas presenciais ou online.

Funcionalidades PlanejadasMatchmaking Inteligente: Filtros por localização, estilo de jogo, sistema e nível de experiência para encontrar o grupo ideal.Gestão de Eventos Híbridos: Sistema de RSVP, notificações automáticas e geolocalização para encontros físicos.Catálogo Interativo: Base de dados com manuais, vídeos e avaliações de jogos integrada às mesas ativas.Rede Social e Reputação: Perfis personalizados com histórico de partidas e sistema de feedback entre jogadores.

Metodologia AplicadaPesquisa de mercado e análise de concorrência (Roll20, Discord, etc.).Validação com usuários reais através de formulários e entrevistas.Desenvolvimento de Modelo de Negócios (Business Model Canvas).
